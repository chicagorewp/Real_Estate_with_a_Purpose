# REAL ESTATE WITH PURPOSE

## Local

* ```git clone``` [https://github.com/the-difference-engine/real-estate-with-purpose.git]

* ```bundle install```

* ```rake db:create```

* ```rake db:migrate```

* To start server: ```rails server```

* To run tests: ```rspec spec```

* Rails version 5.0

### Environment Variables

* ```touch .env```
